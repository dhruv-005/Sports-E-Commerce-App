🏏 Sports E-Commerce Flutter App
Sports E-Commerce is a Flutter-based mobile application designed for managing sports accessories on a single unified platform. The app allows users to authenticate securely, view and manage sports products, and store data locally using SharedPreferences.
A Flutter-based mini e-commerce application focused on a single domain – Sports Accessories, built as part of an academic project to demonstrate REST API integration, JSON parsing, and dynamic UI rendering using GridView/ListView.

---
📌 Project Overview

This application allows users to:

  * Browse sports accessories from a public REST API
  * View product details
  * Add and remove products from the cart
  * See total cart value in real time
  * Switch between Light and Dark themes
---
🎯 Project Objectives

  * Integrate a public REST API into a Flutter application
  * Parse JSON data using Dart model classes
  * Display data using GridView and ListView
  * Implement basic e-commerce functionality
  * Maintain a clean, modular project structure
---
🌐 Public REST API Used

Fake Store API (Free & Public)
https://fakestoreapi.com/products

API Usage

  * HTTP GET request is sent from Flutter app
  * JSON response is received
  * Data is parsed into Dart objects
  * Products are displayed as sports accessories

---
🔄 Application Flow (Real-Time)

  1. App launches → Splash Screen
  2. Home Screen fetches data from REST API
  3. JSON data is parsed into Product models
  4. Products are displayed using **GridView**
  5. User adds/removes items from Cart
  6. UI updates instantly using **Provider state management**

▶️ How to Set Up and Run the Project
✅ Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Android Emulator or Physical Device

Check Flutter installation:

flutter doctor

---

🔽 Step 1: Clone the Repository

git clone https://github.com/dhruv-005/Ecoomerce_Sports_App

---

📁 Step 2: Navigate to Project Directory

cd sports-ecommerce-flutter

---

📦 Step 3: Install Dependencies

flutter pub get

---

▶️ Step 4: Run the Application
	
flutter run
